# Wordpress - Wordpress Headless - Next.js

Plantilla para sitios Wordpress a la medida con WP Headless y Next.js.

### Tecnología
- Next.js
- GraphQL
- Apollo client
- WP admin
- WP Headless
- MySQL


### Servidor de desarrollo 
- MAMP 4.2 -> PHP 7.4
- NGINX

### Accesos
- admin: http://localhost/WordpressReactHeadless/wordpress/wp-login.php
- usuario: admin
- contraseña: admin
